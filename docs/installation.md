# Installation

## Stable release

To install Python NGSI-LD Client, run this command in your terminal:

```sh
uv add python-ngsild-client
```

Or if you prefer to use `pip`:

```sh
pip install python-ngsild-client
```

## From source

The source files for Python NGSI-LD Client can be downloaded from the [Github repo](https://github.com/hyvilo-it-admin/python-ngsild-client).

You can either clone the public repository:

```sh
git clone https://github.com/hyvilo-it-admin/python-ngsild-client
```

Or download the [tarball](https://github.com/hyvilo-it-admin/python-ngsild-client/tarball/main):

```sh
curl -OJL https://github.com/hyvilo-it-admin/python-ngsild-client/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd python-ngsild-client
uv sync
```
